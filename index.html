Navegadores web

Un navegador web (en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incluidos.

Ejemplo de navegadores web

Safari: Safari es un navegador web de código cerrrado desarrollado por Apple. Está disponible para macOS, iPadOS e IOS.

Mozilla Firefoz es un navegador web libre y de código abierto desarrollado por Linux, Android, IOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla.

Google Chrome: Google Chrome es un navegador web de código cerrado desarrollado por google, aunque derivado de proyectos de código abierto.
